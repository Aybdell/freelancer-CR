# FreelanceCRM

A modern CRM platform built for freelancers to manage clients, projects, and invoices in one place.

## Live Demo

https://freelancer-cr.vercel.app/

## Features

* Client Management
* Project Tracking
* Invoice Management
* Dashboard Analytics
* Authentication System
* Responsive Design

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend & Database

* Supabase

### Deployment

* Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/Aybdell/freelancecrm.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000

## Why I Built This

Freelancers often use multiple tools to manage clients, projects, and invoices. FreelanceCRM brings these workflows together into a single platform with a clean and modern interface.
